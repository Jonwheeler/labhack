Sensor Data Visualization Challenge

Participants in this challenge will stream sensor data from a Rasberry Pi or Arduino Yun to Plotly, then download and analyze that data in an interesting way using another Plotly API (MATLAB, Python, R, Julia, etc). For examples of Plotly sensor and telemetry projects, see Plotly’s workshop page: https://plot.ly/workshop/

Entries will be judged based on compelling use and analysis of sensor data obtained from Hardware.

You can learn more about the Arduino API here. You can learn more about Raspberry Pi + Plotly here. You can learn more about the other APIs here.

Awards

  • First Place: Patagonia Plotly Jacket
  • Second Place: Raspberry Pi or an Arduino Yùn with Wifi Shield
